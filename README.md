# 🚀 One Window Smart Academic Portal (UMT)

A comprehensive **Software Engineering project** focused on designing and prototyping a centralized academic management system for **University of Management and Technology (UMT)**.

This project integrates multiple university services into a **single unified platform**, improving efficiency, accessibility, and user experience for students, teachers, and administrators.

---

## 📌 Project Overview

The **One Window Smart Academic Portal** is designed to eliminate fragmented academic systems by providing a **single access point** for all academic operations.

It combines key functionalities of:
- Learning Management System (LMS)
- Student Information System (SIS)
- Enterprise Resource Planning (ERP)

The system is designed using **modern software engineering principles**, including modular architecture and scalable design.
